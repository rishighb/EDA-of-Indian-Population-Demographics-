# EDA-of-Indian-Population-Demographics

This project performs an in-depth Exploratory Data Analysis (EDA) on Indian Census datasets using Python. The objective is to uncover patterns, correlations, and regional disparities across key demographic indicators such as literacy rates, income distribution, urbanization, and population density.

🔍 What This Project Covers

Built a structured data cleaning and preprocessing pipeline to ensure consistency and reliability of census data.

Analyzed 30+ demographic variables across Indian states and union territories.

Identified relationships between literacy, income levels, urbanization rates, and socioeconomic development.

Created clear, interpretable visualizations to highlight regional trends and disparities.

🛠️ Tech Stack

Python

Pandas

Matplotlib

Seaborn

Jupyter Notebook

📊 Key Insights

Strong correlation between literacy rates and urbanization levels in multiple states.

Significant regional differences in income distribution and population growth.

Clear socioeconomic divide between highly urbanized and predominantly rural states.

🎯 Objective

To transform raw census data into meaningful insights that can support data-driven policymaking, research, and academic analysis.
